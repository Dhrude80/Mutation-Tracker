# Mutation-Tracker
Simple mutation tracker tracking magnitude of changes after each mutation.

The DNA string is randomly generated and mutated every iteration ,  causing it to change over time.

---

## Features

- Random string generation
- NumPy based matrix generation
- Difference tracker
- Matplot Lib plotting

---

## Technologies used

- Python
- NumPy
- MatplotLib

## Requirements

Install required libraries before running.
